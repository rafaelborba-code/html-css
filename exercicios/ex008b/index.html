Código-fonte / Pré-formatação

O comando document.getElementById('teste') é escrito em JavaScript. Para representar algo escrito em linguagem de programação, usa-se a tag <code>

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

Para manter um trecho como ele está escrito no código, usa-se a tag <pre>

Citações

Citação simples

Para citações simples e curtas, usa-se a tag <q>

Citações completas

Para citações mais longas e que tenham um parágrafo só para si, usa-se a tag <blockquote>
Eu não quero que vocês estejam esperançosos. Quero que vocês estejam em pânico. Quero que ajam como se a casa estivesse pegando fogo, porque ela está.

Abreviações

Sempre que quiser escrever uma sigla e deixar claro a quem visita o site o que ela significa, use abreviações com a tag <abbr>
