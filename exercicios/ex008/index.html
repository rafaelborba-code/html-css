Principais formatações

Negrito

Para deixar um termo em negrito, basta usar a tag <strong>

Itálico

Para deixar um termo em itálico, basta usar a tag <em>

Texto marcado

Para deixar um termo marcado, como se tivesse usado marca texto, basta usar a tag <mark>
Para mudar a cor do "marca-texto", usa-se o CSS, com o parâmetro style junto do mark.
Exemplo: <mark style="background-color:lime">

Também é possível modificar todas as marcações do script usando também o CSS. Para isso, cria-se uma tag style no cabeçalho(<head>) e se adiciona as informações.

Texto grande e pequeno

Para criar um texto grande e um texto pequeno, usam-se as tags <big> e <small>, respectivamente.
A tag <big> não é mais utilizada, pois não tem um significado.

Texto deletado

É possível marcar um texto como deletado, indicando que ele deve ser lido e não considerado. Para marcá-lo dessa maneira, usa-se a tag <del>

Texto incluído

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag <ins>

Texto sobrescrito

O texto que fica superior ao texto normal, como um expoente. Usa-se a tag <sup>

Texto subscrito

O texto que fica inferior ao texto normal. Usa-se a tag <sub>
